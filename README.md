### What I learned

This project allowed me to practice on font fallbacks and how to deploy simple projects (index.html + styles.css) on Vercel.

```css
@font-face {
  font-family: "Outfit-fallback";
  src: local(Arial);
  ascent-override: 100.55%;
  descent-override: 26.14%;
  line-gap-override: 0%;
  size-adjust: 99.45%;
}
```
